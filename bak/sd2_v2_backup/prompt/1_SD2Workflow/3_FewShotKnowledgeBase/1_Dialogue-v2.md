# Bucket: dialogue
v2.0

## 适用场景

- 双人或多人对白
- 对峙、盘问、施压、问答
- 需要稳定轴线和听者反应的段落
- 声画分离、喜剧快切对话

## 常见结构标签

- `two_person_confrontation`
- `group_dialogue`
- `listener_reaction`
- `table_barrier`
- `axis_sensitive`
- `voice_image_split`（v2 新增）
- `comedy_rhythm`（v2 新增）

## 注入重点

- 先建立空间关系，再进入说话者/听者节奏
- 保持轴线稳定，优先固定镜头或缓推
- 听者反应镜头不能丢
- 微表情要跟对白情绪同步
- **对手说话时切到主角（受力方）**——施力/受力框架

## 密度约束（SD2 引擎安全）

- **单时间片优先一讲一听**
- 多人场景除焦点两人外，其余角色默认**静止或极低动态**
- 肢体接触类对话单独占用时间片

## 代表性 few-shot

### `dialogue_two_person_lowkey_v2`

- `pattern_summary`: 双人低调对峙，主镜建立空间关系，再进入正反打与听者反应
- `camera_bias`: `["固定镜头", "缓慢推进"]`
- `must_cover`: `["axis_stability", "listener_reaction", "micro_expression"]`

<details>
<summary>example_prompt（完整三段式正例）</summary>

```
@图1（沈渡）为主角，青年男性，面容冷峻。@图2（赵令仪）为对手角色，中年女性，神情冷厉。场景为@图3（审讯室），封闭石室，仅一盏油灯悬于二人之间，暖黄灯光从顶部直射，照亮二人面部下半侧，眉弓以上沉入阴影。光线稳定。

0-4s：@图1（沈渡）坐在@图3（审讯室）的石凳上，双手平放于粗木桌面，指尖微微发白，目光低垂注视桌面裂缝。画面左侧的@图2（赵令仪）站立，双臂交叉于胸前，嘴角压着一丝冷笑。镜头固定，中景框住二人与桌面，绝对稳定。4-8s：@图2（赵令仪）身体前倾，右手掌按在桌面上发出沉闷响声，油灯火苗因气流轻微晃动，@图2（赵令仪）的面部暖光随之摇曳。@图1（沈渡）眼皮微抬，瞳孔收缩但身体纹丝不动，下颌肌肉紧绷。镜头固定不动。8-12s：@图1（沈渡）缓缓抬起头，与@图2（赵令仪）目光对视，微张双唇开口说话，声音平静但喉结微微滚动。@图2（赵令仪）后仰半步，双眉微拧，眼角细纹加深，冷笑逐渐消退。镜头缓慢推进至@图1（沈渡）面部近景。12-15s：@图2（赵令仪）右手缓缓从桌面收回，指尖在粗糙桌面划过，目光避开@图1（沈渡），偏向画面右下。@图1（沈渡）维持直视姿态，嘴角极轻微上扬但眼神冷硬。镜头固定，捕捉听者反应。

3D写实动画，4K高清，细节丰富。人物面部稳定不变形，五官清晰，无穿模。双人轴线关系全程保持一致，无跳轴。
```

</details>

**structural_notes**（v2 新增）:
- 时间分配：建立（4s）→ 施压（4s）→ 反击（4s）→ 听者反应（3s），四段接近等分
- 对手施压时（4-8s），主角的微反应（眼皮微抬、瞳孔收缩、下颌紧绷）已经在同一时间片写出——不是空切到主角
- 最后一段是听者反应收尾——对手的退缩比主角的胜利更重要
- 全程只用固定镜头和缓推两种运镜，极度克制

**anti_patterns**（v2 新增）:
- 不要迁移"油灯火苗摇曳"——这是本场景的环境细节，不适用于所有低调室内
- 不要迁移"桌面裂缝"——这是本场景的材质细节
- 不要把四段等分当硬模板——应根据实际对白长度和情绪密度调整

### `dialogue_group_pressure_v2`

- `pattern_summary`: 多人场合中的单人承压，对话重心围绕核心角色逐步收紧
- `camera_bias`: `["固定镜头", "局部缓推"]`
- `must_cover`: `["reaction_contrast", "eye_focus_shift", "spatial_clarity"]`

<details>
<summary>example_prompt（完整三段式正例）</summary>

```
@图1（陆峥）为主角，青年军官，神情克制。@图2（韩司令）坐于长桌正中，年长男性，目光沉稳。@图3（参谋甲）和@图4（参谋乙）分坐两侧。场景为@图5（作战会议室），长桌横贯画面，头顶日光灯管发出冷白光线，均匀照亮桌面与人物，无明显阴影方向。光线稳定。

0-4s：@图5（作战会议室）全景，长桌两侧坐满军官，画面正中的@图2（韩司令）双手交叉搁在桌面，面无表情注视画面右侧末端站立的@图1（陆峥）。@图3（参谋甲）和@图4（参谋乙）同时侧头看向@图1（陆峥）。镜头固定，全景构图。4-8s：@图1（陆峥）立正站姿，双手贴裤缝，喉结滚动一次后开口陈述，语速平稳但额头有细密汗珠。@图2（韩司令）右手食指在桌面无声敲击。镜头缓慢推进至@图1（陆峥）中景。8-12s：@图3（参谋甲）摇头并翻开面前的文件夹，纸页翻动声清晰。@图1（陆峥）目光短暂偏向@图3（参谋甲），随即回正继续陈述，嘴唇微微颤动。镜头固定于@图1（陆峥）近景，景深虚化背景众人。12-15s：@图2（韩司令）抬起右手示意静默，所有人停止动作。@图1（陆峥）闭嘴，下颌收紧，双眼微眯。@图2（韩司令）嘴唇未动但目光从文件抬起直视@图1（陆峥）。镜头固定，双人关系中景。

3D写实动画，4K高清，细节丰富。人物面部稳定不变形，五官清晰，无穿模。多人物面部特征保持一致，严禁跳脸。
```

</details>

**structural_notes**（v2 新增）:
- 焦点收束模式：全景（全员）→ 中景（主角）→ 近景（主角虚化他人）→ 双人关系（上级+主角）
- 主角全程是受力方——陈述、被质疑、被沉默压制
- 非焦点角色（参谋甲乙）只有极低动态（侧头、翻文件），不抢戏
- 最后时间片的权力关系通过"抬手 → 所有人停"展现，不用台词

**anti_patterns**（v2 新增）:
- 不要迁移"军事会议中的身份标识"——这是题材特异信息
- 不要迁移"日光灯管冷白光"——这是特定场景的光源
- 不要在非权力场景中套用"抬手示意静默"的权力手势

### `dialogue_voice_split_fourcut_v1`（v2 新增）

- `pattern_summary`: 声画分离四段切——A 说 → 切 B 反应（A 声音继续）→ 回 A → B 沉默
- `camera_bias`: `["固定镜头"]`
- `must_cover`: `["listener_reaction", "micro_expression", "silence_density"]`

<details>
<summary>example_prompt（完整三段式正例）</summary>

```
@图1（宋衍）为主角（受力方），青年男性，面容冷峻。@图2（陈母）为施力方，中年女性。场景为@图3（客厅），夜间，台灯暖光从画面右侧照亮@图2（陈母）面部，@图1（宋衍）所在的画面左侧光线较暗。光线稳定。

0-4s：@图2（陈母）坐在@图3（客厅）沙发上，身体前倾，双手握紧搁在膝盖上，嘴唇快速开合说着话，眉头紧锁，面部肌肉绷紧。画面左侧的@图1（宋衍）背靠门框站立，双臂交叉于胸前，面部表情平静但下颌肌肉微紧。镜头固定于@图2（陈母）近景。4-7s：镜头切到@图1（宋衍）近景，@图1（宋衍）目光从@图2（陈母）方向缓缓移开看向地面，嘴唇紧抿，喉结缓慢滚动一次，右手手指在左臂上无意识轻叩。光线较暗的环境中@图1（宋衍）面部只有微弱侧光勾勒轮廓。镜头固定。7-10s：镜头回到@图2（陈母），@图2（陈母）说到最后一句，语速放慢，嘴唇动作清晰，说完后双手松开膝盖缓缓后靠沙发背。镜头固定。10-15s：@图1（宋衍）大特写，双眼微眯注视前方虚空，嘴唇微张但未说话，鼻翼轻微翕动一次，呼吸节奏略深。右手从左臂放下垂于体侧，手指缓慢攥紧成拳，指节泛白。持续沉默。镜头固定于@图1（宋衍）大特写。

3D写实动画，4K高清，细节丰富。人物面部稳定不变形，五官清晰，无穿模。
```

</details>

**structural_notes**（v2 新增）:
- **四段切时间分配**：A说（4s）→ B反应（A声音继续）（3s）→ A最后一句（3s）→ B沉默（5s）
- **声画分离段（4-7s）**：画面只写 B 的一个清晰反应状态（目光移开+喉结滚动+手指轻叩），不要太复杂
- **沉默段（10-15s）是全组最重要的镜头**：描写密度不能低于台词段——攥拳指节泛白、鼻翼翕动、呼吸节奏变化
- **整组只用近景和大特写两种景别**，不切中景/远景
- 施力方（陈母）的镜头只负责"交付台词"，受力方（宋衍）的镜头负责"承载情绪"

**anti_patterns**（v2 新增）:
- 不要迁移"台灯暖光从右侧"——光源要匹配实际场景
- 不要迁移"背靠门框"——站位要匹配实际 staging_constraints
- 不要把沉默段缩短到 2-3s——沉默段是情绪核心，必须有足够时长
- 不要在声画分离段写太多动作——一个清晰的反应状态即可
