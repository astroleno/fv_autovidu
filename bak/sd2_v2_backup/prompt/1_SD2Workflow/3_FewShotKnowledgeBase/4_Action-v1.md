# Bucket: action
v1.0

## 适用场景

- 追逐、打斗、冲撞
- 技能释放、爆发性行为
- 强位移、高冲击、强节拍动作

## 常见结构标签

- `chase`
- `duel`
- `skill_release`
- `impact_hit`

## 常见视觉标签

- `high_motion`

## 注入重点

- 节奏通常为“蓄力 → 释放 → 结果”
- 优先保证动作可读性，再追求视觉刺激
- 运镜以跟随、甩镜、升降、环绕为主，但单时间片只能保留一个主运镜
- 多角色时先确保方位和主体关系清楚，再写动态

## 代表性 few-shot

### `action_chase_corridor_v1`

- `pattern_summary`: 强位移追逐，镜头以跟随和方向性压迫感为核心
- `camera_bias`: `["跟随镜头", "短促甩镜"]`
- `must_cover`: `["direction_readability", "speed_boundary", "space_continuity"]`

```
@图1（卫戎）为主角，青年男性，短发凌乱沾血，身穿黑色作训服，右肩撕裂露出内衬。@图2（追兵甲）为追击方，全身黑色战术装备，面罩遮脸。场景为@图3（地下通道），混凝土墙壁，顶部应急灯间隔闪烁发出冷白频闪光，地面有浅层积水。
```

### `action_duel_release_v1`

- `pattern_summary`: 对峙后的爆发，先压住气氛，再在释放瞬间拉高动能
- `camera_bias`: `["固定起势", "快速推进或环绕"]`
- `must_cover`: `["action_readability", "impact_clarity", "timing_punch"]`

<details>
<summary>example_prompt（完整三段式正例）</summary>

```
@图1（钟馗）为主角，威严男性，浓眉虬髯，身穿朱红色道袍，腰系黑色宽腰带，右手持桃木剑。@图2（海妖）为对手，人形态但皮肤泛青鳞光，指尖有蹼膜，湿发贴面。场景为@图3（码头废墟），夜间，断裂的木桩和倾斜的船板，月光从正上方投下冷蓝硬光，海水反光在残骸表面形成不规则光斑。

0-4s：@图3（码头废墟）中景，画面左侧的@图1（钟馗）持桃木剑横于胸前，剑身在月光下泛出暗红木质光泽，道袍下摆因海风向右飘动。画面右侧的@图2（海妖）半蹲于断裂木桩上，十指张开，蹼膜在月光下半透明，身体重心前压。二人对峙静止。镜头固定。4-7s：@图2（海妖）双腿蹬木桩弹射出去，木桩在冲击下断裂飞散，@图2（海妖）身体在空中拧转，右手爪状劈向@图1（钟馗）面部方向。海水因弹射溅起。@图1（钟馗）桃木剑从横置转为右斜上挡，朱红道袍衣袖因出剑动作向后鼓起。镜头快速推进至二人交锋点。7-11s：@图1（钟馗）桃木剑与@图2（海妖）右爪正面碰撞，接触点爆出一圈微弱暗红光晕沿剑身扩散。@图2（海妖）身体被弹开后退两步，左脚踩入浅水，水花溅至小腿。@图1（钟馗）右臂因碰撞反震微颤，随即稳住剑势，虬髯因冲击波轻微散开。镜头环绕至@图1（钟馗）侧后方。11-15s：@图1（钟馗）向前踏出一大步，桃木剑从上至下劈下，道袍因大幅动作完全展开如翼。@图2（海妖）后仰避开剑锋，剑尖在@图2（海妖）面前三寸处划过，斩断数根湿发。海风将断发吹散。镜头固定于侧面中景。

3D写实动画，4K高清，细节丰富。人物面部稳定不变形，五官清晰，无穿模。打击碰撞瞬间动作可读，主体方向关系清晰。
```
