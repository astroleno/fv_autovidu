# 输入字段契约

`scripts/generate.mjs` 接受 4 个 flag，下表是它们的完整契约。

## 契约表

| flag | 必填 | 类型 | 说明 |
|------|------|------|------|
| `--script` | 是 | 路径或文本 | 剧本内容。若是路径，读文件；否则当作内联文本 |
| `--brief` | 是 | 字符串 | 一句话导演简报（见下文格式） |
| `--assets-file` | 是 | JSON 文件路径 | 资产列表 JSON（见 [assets-schema.md](assets-schema.md)） |
| `--slug` | 否 | 字符串 | 产物目录名；缺省时用 `sd2-<timestamp>` |

## `--script` 详解

两种写法：

1. **路径模式**（推荐）：`--script public/script/test/e1.md`
2. **内联模式**：`--script "第一场 医院走廊 …"`（内容带空格请加引号）

脚本自动识别：若值包含换行或 `\n`，按内联处理；否则尝试当作路径读文件，读不到再退回内联。

## `--brief` 详解

**一句话**包含以下关键信息（顺序不限）：

| 信息 | 必要性 | 示例片段 | 说明 |
|------|--------|----------|------|
| 时长 | 强烈推荐 | "单集总时长 120 秒" | 决定 Block 数与时长分配 |
| 目标镜头数 | 推荐 | "目标镜头数约 60" | 决定剪辑密度 |
| 题材 | 推荐 | "都市情感短剧" | 影响 EditMap 结构加权 |
| 渲染风格 | 推荐 | "真人电影风格" | 写入 renderingStyle |
| 美术风格 | 可选 | "冷调偏青，高反差，低饱和" | 写入 artStyle |

**完整示例**：

```
单集总时长 120 秒；目标镜头数约 60。现代都市医疗情感短剧，真人电影风格。冷调偏青，高反差，低饱和。
```

**最小示例**（能跑，但参数会用默认）：

```
单集总时长 90 秒。
```

brief 会原样写入 `edit_map_input.json` 的 `directorBrief` 字段，由 EditMap LLM 自行解析结构化参数（见 `prepare_editmap_input.mjs` 的 `parseBriefStructuredHints`）。

## `--assets-file` 详解

JSON 文件路径，内容必须是一个数组。格式见 [assets-schema.md](assets-schema.md)。

校验失败时 `generate.mjs` 会：

1. 打印首个错误和示例
2. 以 exit code 2 退出

Claude 应把错误原样回给用户并要求修正。

## `--slug` 详解

- 字符集限制：`[a-z0-9-]`，最长 40 字符
- 产物目录：`output/sd2/<slug>/`
- 已存在同名目录时脚本会**覆盖**，并给出警告；若要保留历史请用不同 slug

缺省时自动生成 `sd2-YYYYMMDD-HHmmss`（本地时区）。

## 退出码约定

| exit code | 含义 | Claude 的应对 |
|-----------|------|---------------|
| `0` | 成功，产物完整 | 汇报产物路径 + 关键数字 |
| `2` | 输入校验失败 | 把错误和示例回给用户 |
| `3` | 两次重试都失败 | 读 troubleshooting.md 建议下一步 |
| `4` | 前置条件缺失（`.env` / 依赖文件） | 提示用户检查 `.env` 与工作目录 |
| `1` | 其它未知错误 | 把日志尾部原样回给用户 |
