# 组结构硬约束

**消费者：Director**
**注入条件：always（每次 Director 调用都注入）**
**版本：v1.0**

---

## 硬约束清单

### 1. 组时长约束

| 约束 | 规则 | 来源 |
|------|------|------|
| 组总时长下限 | >= 5s | EditMap 全局规则 |
| 组总时长上限 | <= 16s | EditMap 全局规则 |
| Hook/Cliff 时长上限 | <= 10s | EditMap 首尾组规则 |

### 2. 子镜头（时间片）约束

| 约束 | 规则 | 说明 |
|------|------|------|
| 单子镜头最小时长 | **>= 3s** | Seedance 2.0 单镜头 < 3s 无法承载完整表演或有效运镜变化 |
| 冲击帧例外 | 允许 2s，但**每组最多 1 个** | 用于冲击帧、反应帧等极短镜头 |
| 单子镜头最大时长 | <= 8s（极端允许 10s） | 超长单镜头画面稳定性下降 |
| 每组子镜头数量 | **<= 5 个** | 默认 2-4 个，4 档爆发允许 5 个。**禁止 6 个及以上** |

### 3. 景别连续性约束

| 约束 | 规则 | 说明 |
|------|------|------|
| 禁止连续同景别 | **禁止连续 3 个同景别时间片** | 如连续 3 个特写、连续 3 个中景均违规 |
| 禁止批量标签 | 禁止 `景别：特写x3` / `三个特写快切` | 每个时间片必须独立一行，有独立的景别/角度/运镜 |

### 4. 时间守恒约束

| 约束 | 规则 |
|------|------|
| 时间片首尾相接 | 无空洞、无重叠 |
| 时间片之和 == 组时长 | 所有时间片时长之和必须严格等于 block_index 中的 duration |
| 整数秒 | 所有时间片时长必须为整数秒，禁止小数 |

---

## 自检清单

Director 在完成分镜稿后，必须逐条确认：

- [ ] 每个时间片 >= 3s（2s 例外最多 1 个/组）
- [ ] 每组时间片数量 <= 5
- [ ] 无连续 3 个同景别时间片
- [ ] 时间片首尾相接，总和 == 组时长
- [ ] 所有时间片时长为整数秒
- [ ] 每个时间片独立一行，无批量标签
