# Bucket: action
v2.0

## 适用场景

- 追逐、打斗、冲撞
- 技能释放、爆发性行为
- 强位移、高冲击、强节拍动作
- 一招制敌/碾压式快结（v2 新增）
- 危机快切/紧急爆发（v2 新增）

## 常见结构标签

- `chase`
- `duel`
- `skill_release`
- `impact_hit`
- `instant_defeat`（v2 新增）
- `crisis_burst`（v2 新增）

## 常见视觉标签

- `high_motion`

## 注入重点

- 节奏通常为"蓄力 → 释放 → 结果"
- 优先保证动作可读性，再追求视觉刺激
- 运镜以跟随、甩镜、升降、环绕为主，但单时间片只能保留一个主运镜
- 多角色时先确保方位和主体关系清楚，再写动态

## 代表性 few-shot

### `action_chase_corridor_v2`

- `pattern_summary`: 强位移追逐，镜头以跟随和方向性压迫感为核心
- `camera_bias`: `["跟随镜头", "短促甩镜"]`
- `must_cover`: `["direction_readability", "speed_boundary", "space_continuity"]`

<details>
<summary>example_prompt（完整三段式正例）</summary>

```
@图1（卫戎）为主角，青年男性，右肩带伤。@图2（追兵甲）为追击方，动作凶狠，面部特征被阴影遮挡。场景为@图3（地下通道），混凝土墙壁，顶部应急灯间隔闪烁发出冷白频闪光，地面有浅层积水。光线稳定。

0-3s：@图3（地下通道）纵深全景，@图1（卫戎）从画面深处向镜头方向全力奔跑，右手扶墙借力转弯，脚底踏入积水溅起水花。应急灯频闪在墙面投下断续的白色光斑。镜头向后跟随@图1（卫戎），运动轨迹平滑。3-6s：@图1（卫戎）跑过一处交叉路口，身体向左急转，左手抓住拐角墙壁边缘，指尖在混凝土上刮出白痕。@图2（追兵甲）的身影从画面右后方出现，奔跑速度快于@图1（卫戎）。镜头短促甩向左侧跟随@图1（卫戎）转弯方向。6-10s：@图1（卫戎）在狭窄通道内冲刺，喘息急促，右肩伤口处因摆臂幅度变大而被拉扯。@图2（追兵甲）的脚步声越来越近，应急灯在@图2（追兵甲）头部轮廓上反射出冷白光点。镜头跟随@图1（卫戎）背部中景，微带追赶偏移。10-14s：@图1（卫戎）猛然侧身闪入一扇半开的铁门后方，铁门在惯性下发出沉重撞击声。@图2（追兵甲）冲过铁门位置未能刹住，脚底在积水地面打滑。镜头固定于铁门处，@图2（追兵甲）从画面中快速穿过。

3D写实动画，4K高清，细节丰富。人物面部稳定不变形，五官清晰，无穿模。运动模糊集中在四肢末端，主体躯干保持清晰。
```

</details>

**structural_notes**（v2 新增）:
- 追逐四段式：全力奔跑（建立速度）→ 急转弯（方向变化）→ 追距缩短（压力升级）→ 机会利用（侧身闪入）
- 运镜从跟随→甩镜→跟随→固定，节奏与逃跑路线同步
- 追兵始终是"背景压力"——不单独给大特写，只通过声音和身影存在
- 物理细节集中在与环境的交互：扶墙、踩水、刮混凝土、积水打滑

**anti_patterns**（v2 新增）:
- 不要迁移"地下通道""应急灯""积水"——环境要匹配实际
- 不要迁移"追击方的战术化造型细节"——这是题材特异外观信息
- 不要在追逐中加入对话——追逐的张力在于纯物理行为
- 不要给追兵太多描写——追逐场景的焦点是逃跑者

### `action_duel_release_v2`

- `pattern_summary`: 对峙后的爆发，先压住气氛，再在释放瞬间拉高动能
- `camera_bias`: `["固定起势", "快速推进或环绕"]`
- `must_cover`: `["action_readability", "impact_clarity", "timing_punch"]`

<details>
<summary>example_prompt（完整三段式正例）</summary>

```
@图1（钟馗）为主角，威严男性，浓眉虬髯，右手持桃木剑。@图2（海妖）为对手，人形态但皮肤泛青鳞光，指尖有蹼膜，湿发贴面。场景为@图3（码头废墟），夜间，断裂的木桩和倾斜的船板，月光从正上方投下冷蓝硬光，海水反光在残骸表面形成不规则光斑。光线稳定。

0-4s：@图3（码头废墟）中景，画面左侧的@图1（钟馗）持桃木剑横于胸前，剑身在月光下泛出暗红木质光泽，下摆因海风向右飘动。画面右侧的@图2（海妖）半蹲于断裂木桩上，十指张开，蹼膜在月光下半透明，身体重心前压。二人对峙静止。镜头固定。4-7s：@图2（海妖）双腿蹬木桩弹射出去，木桩在冲击下断裂飞散，@图2（海妖）身体在空中拧转，右手爪状劈向@图1（钟馗）面部方向。海水因弹射溅起。@图1（钟馗）桃木剑从横置转为右斜上挡，衣袖因出剑动作向后鼓起。镜头快速推进至二人交锋点。7-11s：@图1（钟馗）桃木剑与@图2（海妖）右爪正面碰撞，接触点爆出一圈微弱暗红光晕沿剑身扩散。@图2（海妖）身体被弹开后退两步，左脚踩入浅水，水花溅至小腿。@图1（钟馗）右臂因碰撞反震微颤，随即稳住剑势，虬髯因冲击波轻微散开。镜头环绕至@图1（钟馗）侧后方。11-15s：@图1（钟馗）向前踏出一大步，桃木剑从上至下劈下，因大幅动作衣摆完全展开。@图2（海妖）后仰避开剑锋，剑尖在@图2（海妖）面前三寸处划过，斩断数根湿发。海风将断发吹散。镜头固定于侧面中景。

3D写实动画，4K高清，细节丰富。人物面部稳定不变形，五官清晰，无穿模。打击碰撞瞬间动作可读，主体方向关系清晰。
```

</details>

**structural_notes**（v2 新增）:
- 对决四段式：对峙蓄力（静止）→ 先手进攻（弹射+空中拧转）→ 碰撞+反弹（力量对比）→ 反击（劈剑+闪避）
- 运镜递进：固定（蓄力）→ 快速推进（碰撞点）→ 环绕（展示结果）→ 固定（最终姿态）
- 物理反馈是动作可读性的关键：木桩断裂、水花溅起、反震微颤、断发吹散
- 每个时间片只有一个主动作+一个反应，不堆砌

**anti_patterns**（v2 新增）:
- 不要迁移"桃木剑""海妖""蹼膜"——角色和武器要匹配实际
- 不要迁移"码头废墟""月光冷蓝"——场景和光源要匹配
- 不要在非特效场景写"光晕沿剑身扩散"——只增强原剧本已有的特效
- 不要在单时间片内写多个碰撞——一次碰撞，看清结果

### `action_instant_defeat_v1`（v2 新增）

- `pattern_summary`: 碾压式快结——一招制敌，节奏极快，重点是结果而非过程
- `camera_bias`: `["固定起势", "快速推进"]`
- `must_cover`: `["timing_punch", "impact_clarity", "reaction_contrast"]`

<details>
<summary>example_prompt（完整三段式正例）</summary>

```
@图1（季鸿）为主角，青年男性，面容冷淡，双手插兜。@图2（混混头）为对手，壮硕男性，手持铁管。场景为@图3（小巷），夜间，巷道尽头路灯发出暖黄光，从@图1（季鸿）身后照射形成逆光轮廓，@图2（混混头）面部被正面照亮。光线稳定。

0-4s：@图3（小巷）中景，画面右侧的@图2（混混头）举起铁管指向画面左侧的@图1（季鸿），嘴唇快速开合。@图1（季鸿）双手仍插在兜里，身体重心未移，目光平视@图2（混混头），面部毫无表情变化。路灯逆光在@图1（季鸿）肩部和头发边缘勾勒出暖黄轮廓线。镜头固定。4-6s：@图2（混混头）挥起铁管向@图1（季鸿）头部劈下，铁管划过空气。@图1（季鸿）右手从兜中抽出，以极快速度侧身闪过铁管，右手掌直接拍在@图2（混混头）持铁管的手腕上。铁管脱手飞出。镜头快速推进至二人交手点。6-10s：@图2（混混头）右手腕下垂无法发力，左手本能护住手腕，身体弯腰后退。@图1（季鸿）右手收回重新插入兜中，面部表情始终没有变化，身体恢复直立姿态。铁管落地发出金属撞击声在巷道中回响。镜头固定于@图1（季鸿）中景。10-14s：@图2（混混头）跌坐在地，抬头看@图1（季鸿），嘴唇发颤。@图1（季鸿）低头俯视@图2（混混头），逆光使面部处于阴影中看不清表情，仅下巴轮廓清晰。路灯暖光在@图1（季鸿）肩部持续形成轮廓线。镜头固定于低角度仰拍。

3D写实动画，4K高清，细节丰富。人物面部稳定不变形，五官清晰，无穿模。打击瞬间动作可读。
```

</details>

**structural_notes**（v2 新增）:
- 碾压四段式：蔑视（双手插兜不动）→ 一击（2s 极快解决）→ 回位（面无表情收手）→ 威压俯视（低角度仰拍）
- 核心技法是**反差**：对手的嚣张 vs 主角的冷淡，对手的铁管 vs 主角的徒手，对手的跌坐 vs 主角的双手插兜
- 打击时间片极短（4-6s 只有 2 秒）——碾压不需要过程，结果就是过程
- 最后的仰拍逆光是"权力构图"——看不清表情比能看清更有威慑力

**anti_patterns**（v2 新增）:
- 不要迁移"铁管""小巷""路灯"——武器和场景要匹配实际
- 不要迁移"双手插兜"——这是特定角色的性格动作
- 不要在势均力敌的对决中用碾压模式——碾压模式只适用于实力差距悬殊的场景
- 不要在打击时间片写太多动作——碾压的核心是"快"，一个动作解决
