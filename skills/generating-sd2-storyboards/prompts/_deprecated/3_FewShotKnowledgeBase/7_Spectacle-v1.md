# Bucket: spectacle（视觉奇观与福利）
v1.0

## 定位

为 SD2Director 和 SD2Prompter 提供**视觉奇观/福利/特效爆发**场景的结构化拍摄模板。这些场景是短剧的核心付费驱动力——用户为"钱镜头"停留和付费。

> **商业价值**：美感展示、男频视觉福利、特效释放是短剧观众停留和付费的直接驱动力。缺少显式 SOP 时，模型要么概括为一句话（"女主很美"），要么脑补过度（添加不存在的服装细节）。

## 适用场景

- 美感展示（beauty_reveal）
- 男频视觉福利（fan_service）
- 特效释放（vfx_release）
- 身体接触/暧昧（intimate_contact）

## 常见结构标签

- `body_scan`
- `beauty_showcase`
- `vfx_buildup`
- `intimate_proximity`
- `skill_release`

## 常见视觉标签

- `backlit_rim`
- `shallow_dof`
- `golden_hour`
- `impact_flash`

## 注入重点

- **纯物理描述，禁止比喻**——铁律 1 在 spectacle 场景中尤为重要
- **必须详写不可概括**——"女主很美""画面震撼"是严重错误
- **禁止描写具体服装**——外观由参考图定义
- 每个子型有独立的阶段模板（phase_template），SD2Director 按模板分配时间片

## 接口边界说明

- `spectacle` 只存在于 **few-shot 检索层**（`scene_bucket` / `scene_archetype`）
- `sd2_scene_type` **不新增枚举**，仍保持 `文戏 / 武戏 / 混合`
- 当一个 Block 同时具有动作或情绪属性时，`sd2_scene_type` 继续按现有逻辑填写，`spectacle` 只负责补充"如何拍出奇观感/付费感"的检索模式

## 子型 1：`beauty_reveal`（美感展示）

**触发条件**：女性角色需要被唯美呈现的场景——下车、走路、躺卧、沐浴后、状态焕新后。

**对应 `scene_archetype`**: `beauty_reveal`

### 阶段模板

| 阶段 | 时长范围 | 内容 | camera_intent | 关键要素 |
|------|---------|------|--------------|---------|
| `scan` | 5-8s | 从局部特写开始极缓慢扫描，揭示式入画 | `tilt_up` / `tilt_down`（站/坐时）; `dolly_along_axis`（躺时俯拍推轨） | 浅景深背景虚化；每个部位停留 2-3s；扫描速度极慢 |
| `hold` | 5-8s | 定格近景 + 台词或动作 | `static` / `slow_push` | 环境元素与身体的物理互动 |

### 扫描路径选项

- 手指→手臂→锁骨→脖颈→侧颜
- 脚踝→小腿→膝盖→大腿→腰部→上半身

### 氛围互动词汇库

- 发丝飘动 / 发丝从肩头滑落拉出弧线
- 别发到耳后
- 阳光勾勒轮廓 / 逆光勾勒身体轮廓线
- 微风吹动长发

### 铁律（本子型专用）

1. 所有描写必须用**纯物理描述**，禁止比喻
2. 必须详写不可概括（"女主很美"是严重错误）
3. 禁止描写具体服装——外观由参考图定义

### 代表性 few-shot

#### `spectacle_beauty_reveal_v1`

- `pattern_summary`: 极缓慢身体扫描→定格近景，揭示式入画
- `camera_bias`: `["tilt_up", "固定镜头"]`
- `must_cover`: `["scan_path_template", "physical_description_discipline", "atmosphere_interaction_vocabulary"]`

<details>
<summary>example_prompt（完整三段式正例）</summary>

```
@图1（顾清澜）为主角，年轻女性，神情放松。场景为@图2（庭院），清晨，阳光从画面右后方低角度射入，在@图1（顾清澜）身体左侧勾勒出暖黄轮廓线，背景庭院植物虚化为柔和色块。光线稳定。

0-3s：@图1（顾清澜）的手指特写，右手自然垂于身侧，手指修长，指尖微微弯曲，阳光在手背上形成细长光斑。浅景深，背景完全虚化。镜头从手指开始极缓慢向上移动。3-6s：镜头沿@图1（顾清澜）手臂缓慢上移，经过手腕、前臂，到达肩部和锁骨线条，锁骨处阳光形成一道细窄的高光。发丝从肩头垂落，一缕发丝因微风从肩上滑落拉出弧线。镜头继续缓慢上移。6-9s：镜头到达@图1（顾清澜）脖颈和侧颜，下颌线条清晰，耳后有一缕散发。@图1（顾清澜）微微侧头，阳光从右后方在面部右侧形成暖黄轮廓光，左侧面部处于柔和阴影中。嘴角带着自然的放松弧度。镜头停在侧颜近景。9-14s：@图1（顾清澜）转头面向镜头方向，长发因转头甩动后落在另一侧肩上，目光平视，瞳孔清澈。右手抬起将一缕额前散发别到耳后，手指经过额头、太阳穴、耳廓时动作缓慢流畅。微风吹动背后散落的长发。镜头固定于@图1（顾清澜）正面近景。

3D写实动画，4K高清，细节丰富。人物面部稳定不变形，五官清晰，无穿模。浅景深背景虚化，人物主体始终锐利。
```

</details>

**structural_notes**:
- 扫描路径：手指→手臂→锁骨→脖颈→侧颜→正面，完整的从下到上揭示
- 每个部位停留 2-3s——速度极慢是美感的关键
- 环境互动只有三种：阳光轮廓光、发丝滑落弧线、微风吹发——极克制
- 最后"别发到耳后"是收尾动作——给观众一个完整的面部呈现

**anti_patterns**:
- 禁止写具体服装（"白色连衣裙""丝绸吊带"等）——外观由参考图定义
- 禁止写"皮肤白皙如玉""眼波流转"——比喻会被 AI 字面执行
- 不要在扫描过程中加入对话或动作戏——扫描段是纯视觉享受
- 不要迁移"庭院""清晨阳光"——场景和光源要匹配实际

## 子型 2：`fan_service`（男频视觉福利）

**触发条件**：女性角色下车/弯腰/躺卧/居家松弛状态/沐浴后/被救治/身体接触/近距离暧昧。

**对应 `scene_archetype`**: `fan_service`

### 阶段模板

| 阶段 | 时长范围 | 内容 | camera_intent |
|------|---------|------|--------------|
| `body_scan` | 3-5s | 身体扫描镜头 | `tilt_up` / `tilt_down` / `dolly_along_axis` |
| `detail_closeup` | 2-3s | 局部特写停留 | `static` |
| `atmosphere_hold` | 4-6s | 氛围互动 + 浅景深 | `slow_push` / `static` |

### 氛围互动词汇库

- 发丝从肩头滑落拉出弧线
- 阳光从侧后方勾勒身体轮廓
- 微风吹动长发
- 弯腰时逆光勾勒身体轮廓线

### 铁律（本子型专用）

1. 纯物理描述，禁止比喻
2. 绝对不可概括为一句话
3. 禁止描写具体服装
4. 救治/查看伤口场景必须有男主 POV 视角镜头（角色躺时用俯拍推轨）
5. 同一角色连续不超过两个切镜，必须穿插反应镜头
6. 男女近距离时强调距离感：发丝交织、呼吸可闻、肌肤接近

### 触发场景速查

- 女性角色下车（发丝/身体轮廓/步伐落地声）
- 女性角色躺卧/倒地（长发铺散+身体曲线+光斑流淌）
- 男主救治/查看女主身体（POV 俯拍推轨）
- 男女身体接触（升格描写+接触部位局部特写）
- 女性弯腰/蹲下（逆光轮廓+发丝滑落弧线）

## 子型 3：`vfx_release`（特效释放）

**触发条件**：剧本中有法术、异能、超自然现象需要视觉呈现。

**对应 `scene_archetype`**: `vfx_release`

### 阶段模板

| 阶段 | 时长范围 | 内容 | camera_intent |
|------|---------|------|--------------|
| `charge` | 3-4s | 蓄力：近景→特效从小到大逐层展开 | `slow_pull`（缓慢后退揭示全貌） |
| `burst` | 4-5s | 释放：特效爆发，机位急速运动跟随 | `crane_up`（追流光）或 `rapid_push`（躲冲击波） |
| `impact` | 2-3s | 大远景展示特效对环境的影响 | `static`（极宽画幅） |
| `return` | 3-4s | 切回角色大特写，从壮阔回归安静 | `static`（特写） |

### 环境影响示例词汇

- 声波冲击树林 / 地面龟裂 / 云海涟漪
- 水面掀起环形波纹 / 周围物体被推开

### 铁律（本子型专用）

1. 只增强原剧本已有的特效场景，**不凭空添加新特效**
2. 蓄力→释放→环境影响→角色收尾四阶段不可缺
3. 收尾阶段是情绪落点，不能被特效淹没

### 代表性 few-shot

#### `spectacle_vfx_release_v1`

- `pattern_summary`: 蓄力→释放→环境冲击→角色收尾四阶段
- `camera_bias`: `["slow_pull", "crane_up", "固定镜头"]`
- `must_cover`: `["phase_structure", "impact_clarity", "atmosphere_interaction_vocabulary"]`

<details>
<summary>example_prompt（完整三段式正例）</summary>

```
@图1（凌霄）为主角，青年男性，神情专注，双手结印。场景为@图2（山崖），夜间，月光从正上方投下冷蓝硬光，崖顶岩石表面被照亮，四周是浓黑的山谷深渊。光线稳定。

0-4s：@图1（凌霄）近景，双手在胸前结印，手指交错，指缝间开始泛出微弱的淡蓝色光点。光点从指缝向外扩散，沿手臂向上蔓延。@图1（凌霄）面部表情专注，双眼微闭，额头微微皱起。月光在@图1（凌霄）肩部形成冷蓝边缘光。镜头从@图1（凌霄）近景缓慢后退。4-8s：@图1（凌霄）双手猛然向前推出，手掌张开，淡蓝色光从掌心爆射而出形成光柱，光柱直径从掌心宽度急速扩展至两米。光柱沿途空气扭曲产生热浪纹理。@图1（凌霄）长发被冲击波向后掀起，衣摆猛烈翻飞。崖顶碎石被气浪推出崖缘。镜头急速上升跟随光柱方向。8-10s：大远景，光柱冲击对面山体，碰撞点爆出巨大的淡蓝色光环向四周扩散。山体表面岩石碎裂，碎片向外飞溅。光环经过之处云雾被推散形成环形空洞。崖顶的@图1（凌霄）在画面左下角呈小型剪影。镜头固定于极宽画幅。10-14s：@图1（凌霄）大特写，双手缓缓放下垂于身侧，手指微颤。面部从专注逐渐放松，双眼睁开，瞳孔中倒映着远处山体的淡蓝色余光。呼吸加深，胸廓起伏。月光恢复冷蓝稳定照明。镜头固定于@图1（凌霄）面部特写。

3D写实动画，4K高清，细节丰富。人物面部稳定不变形，五官清晰，无穿模。特效光效不淹没角色主体。
```

</details>

**structural_notes**:
- 四阶段严格遵守：蓄力（4s）→ 释放（4s）→ 环境影响（2s）→ 角色收尾（4s）
- 蓄力阶段用缓慢后退揭示——从近景开始让观众看到细节，后退后看到全貌
- 释放阶段用急速上升——跟随特效方向，强化冲击感
- **收尾阶段（10-14s）是情绪落点**——从壮阔回归安静，描写密度不低于释放段
- 大远景中主角以"小型剪影"存在——不消失但不抢特效视觉主体

**anti_patterns**:
- 不要凭空添加新特效——只增强剧本已有的特效
- 不要迁移"淡蓝色光""结印"——特效颜色和释放方式要匹配实际 Block 的 vfx 描述
- 不要在收尾段继续写特效——收尾是"人"不是"效"
- 不要省略环境影响阶段——壮阔感来自特效对环境的物理影响

## 子型 4：`intimate_contact`（身体接触/暧昧）

**触发条件**：剧本中有男女身体接触、近距离暧昧、拥抱、握手等场景。

**对应 `scene_archetype`**: `intimate_contact`

### 运镜要点

- 关键瞬间使用升格描写（2-4s 实际时间内极慢动作描写）
- 接触部位局部特写
- 交替女主身体扫描（站着 `tilt_up` / `tilt_down`，躺着 `dolly_along_axis`）
- 发丝飞扬、衣袂展开等慢动作微细节

### 距离感词汇库

- 发丝交织
- 呼吸可闻
- 肌肤接近但未触碰
- 指尖触碰瞬间的微颤

### 铁律（本子型专用）

1. 纯物理描述，禁止比喻
2. 禁止描写具体服装
3. 接触瞬间必须有局部特写
4. 强调距离感——"将触未触"比"已经触碰"更有张力
