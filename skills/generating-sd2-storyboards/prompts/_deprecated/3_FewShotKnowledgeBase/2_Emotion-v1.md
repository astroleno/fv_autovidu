# Bucket: emotion
v1.0

## 适用场景

- 单人独角戏
- 沉默反应、凝视、自我消化
- 情绪递进但无大动作场面

## 常见结构标签

- `single_subject`
- `gaze_hold`
- `material_anchor`
- `interior_pressure`

## 常见补强目标

- `micro_expression`
- `breathing_detail`
- `eye_focus_shift`
- `restraint`
- `slow_push`

## 注入重点

- 节奏偏克制，优先 2-4 个时间片
- 用手部、服装、呼吸、视线细节承载情绪变化
- 运镜以固定镜头、缓推、轻微漂移为主
- 重点不是“发生了什么”，而是“角色如何感受”

## 代表性 few-shot

### `emotion_awakening_lowkey_v1`

- `pattern_summary`: 单主体觉醒，先建立压迫氛围，再推进到认知变化
- `camera_bias`: `["缓慢推进", "固定镜头"]`
- `must_cover`: `["micro_expression", "material_interaction", "slow_push"]`

```
@图1（秦狩）为主角，年轻皇子，剑眉星目，束发略散，身穿华贵真丝睡袍。场景为@图2（寝宫），夜间灯火，古色古香的皇家寝宫，幽暗压抑，烛火摇曳。一束冷青色窄光从画面左侧切入，高反差硬切，右侧沉入深青浓重阴影。

0-5s：@图2（寝宫）全景，帷幔垂帘虚化前景，锦缎软榻位于画面中央偏右，@图1（秦狩）在软榻上猛然惊坐起，睡袍衣襟因猛烈动作散开翻飞，丝绸被面留有深深褶皱。帷幔轻微晃动。镜头从寝宫全景缓慢向@图1（秦狩）推进。5-9s：@图1（秦狩）惊恐地环顾四周陌生的环境，眼神快速扫视左右，嘴唇微张低声惊呼，颧骨上冷白高光硬切分明，面部肌肉紧绷。镜头固定，绝对稳定。9-15s：@图1（秦狩）缓缓伸出双手，低头凝视自己年轻的双手，手背肤色偏冷苍白，指节清晰，瞳孔微微颤动，表情从震惊渐变为不可置信。镜头缓慢推进至@图1（秦狩）双手与面部的近景。

3D写实动画，4K高清，细节丰富，粗颗粒胶片质感。人物面部稳定不变形，五官清晰，无穿模，面部表情自然过渡。
```

### `emotion_silent_break_v1`

- `pattern_summary`: 角色强忍情绪，表面静止，内部断裂逐步外显
- `camera_bias`: `["固定镜头", "轻微漂移"]`
- `must_cover`: `["breathing_detail", "eye_focus_shift", "restraint"]`

<details>
<summary>example_prompt（完整三段式正例）</summary>

```
@图1（苏晚）为主角，年轻女性，散发披肩，面色苍白，身穿素白棉质睡裙。场景为@图2（病房），深夜无人的单人病房，窗外城市灯光透过百叶窗在墙面投下均匀横纹，冷蓝色调。床头监护仪屏幕微光映照@图1（苏晚）面部右侧。

0-5s：@图1（苏晚）坐在@图2（病房）床沿，双脚悬空，双手握着一张折叠的信纸，手指关节发白。目光固定在信纸上不动，胸廓极缓慢起伏，呼吸浅而克制。镜头固定，中景侧面取景。5-10s：@图1（苏晚）右手拇指无意识地摩挲信纸折痕，纸面因反复折叠已起毛边。视线从信纸缓缓移开，偏向窗外百叶窗的光纹，瞳孔焦距变远发散，嘴唇紧闭两侧肌肉微微凹陷。监护仪微光在面部右侧持续脉动。镜头轻微向右漂移。10-15s：@图1（苏晚）下唇突然颤抖一次随即咬住，鼻翼翕动，眼眶泛红但无泪水落下。左手缓缓松开信纸，信纸边角轻微卷曲。右手抬起覆盖住口鼻，手指微微颤抖。镜头固定于面部近景。

3D写实动画，4K高清，细节丰富。人物面部稳定不变形，五官清晰，无穿模。面部微表情过渡自然，无跳帧式突变。
```

</details>
