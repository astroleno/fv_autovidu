# Bucket: memory
v1.0

## 适用场景

- 回忆、闪回
- 非现实时间线切入
- 当前现实被过去片段打断或覆盖

## 常见结构标签

- `flashback`
- `memory_fragment`
- `past_overlay`
- `recall_trigger`

## 常见视觉标签

- `non_physical_space`

## 注入重点

- 优先把现实线与记忆线区分清楚
- 允许非物理空间、暗角、柔焦、碎片化遮挡等表达
- 节奏通常短于现实段落，但需要清晰的切入和退出

## 代表性 few-shot

### `memory_fragment_cold_v1`

- `pattern_summary`: 由现实触发记忆碎片，画面边缘收紧并逐步脱离现实空间
- `camera_bias`: `["轻微漂移", "缓慢横移"]`
- `must_cover`: `["transition_boundary", "memory_texture", "return_path"]`

<details>
<summary>example_prompt（完整三段式正例）</summary>

```
@图1（许砚）为主角，青年男性，面容疲惫，黑眼圈明显，身穿深色翻领毛衣。@图2（许砚-少年）为记忆中的角色形象，少年面庞，穿白色校服。场景为@图3（书房），当前现实空间，夜间台灯光照亮桌面局部，其余空间沉入暗灰。记忆空间为非物理背景，纯白过曝光场，边缘渐暗。

0-4s：@图3（书房）中景，@图1（许砚）坐在书桌前，右手握着一枚旧钥匙，拇指反复摩挲钥匙齿槽。台灯暖黄光照亮@图1（许砚）手部与桌面，面部上半部分处于弱光中。@图1（许砚）目光集中在钥匙上，瞳孔焦距变近。镜头固定。4-8s：画面边缘开始收暗，暗角从四角向中心缓慢收拢。@图3（书房）的桌面、书架逐渐模糊失焦，仅@图1（许砚）手中的钥匙保持清晰。画面整体饱和度下降，色调向冷灰偏移。@图1（许砚）面部表情凝固不动。镜头轻微向下漂移至手部特写。8-12s：画面完全脱离@图3（书房），背景转为非物理的纯白过曝光场，边缘带柔和暗角。@图2（许砚-少年）从画面左侧走入，白色校服在过曝背景中边缘微微发光，面部特征清晰但周围空间无物理参照。@图2（许砚-少年）右手也握着同一枚钥匙，低头微笑。轻微柔焦笼罩整个画面。镜头缓慢横移跟随@图2（许砚-少年）。12-15s：画面边缘的暗角快速收缩加重，@图2（许砚-少年）的形象逐渐模糊消散。背景从纯白光场重新浮现@图3（书房）的桌面与书架轮廓，饱和度回升。@图1（许砚）手中的钥匙依然清晰，右手指尖停止摩挲。镜头固定回到书房中景。

3D写实动画，4K高清，细节丰富。人物面部稳定不变形，五官清晰，无穿模。现实与记忆空间的过渡平滑自然，无硬切闪烁。记忆段柔焦与低饱和全程一致。
```

### `memory_reveal_insert_v1`

- `pattern_summary`: 回忆不只是氛围补充，而是承载信息揭示
- `camera_bias`: `["固定镜头", "局部缓推"]`
- `must_cover`: `["fact_delivery", "timeline_separation", "emotional_echo"]`

<details>
<summary>example_prompt（完整三段式正例）</summary>

```
@图1（顾南衣）为主角，青年女性，碎发遮额，当前着黑色风衣。@图2（顾南衣-幼年）为记忆中的角色形象，七八岁女童，齐刘海，穿浅粉色连衣裙。@图3（父亲）为记忆中出现的角色，中年男性，圆框眼镜，白衬衫卷袖。场景当前为@图4（天台），夜间城市灯光远景，风大。记忆空间为@图5（旧宅客厅），午后阳光从左侧窗户射入，暖黄色调，画面带轻微胶片颗粒感。

0-3s：@图4（天台）中景，@图1（顾南衣）站在天台边缘护栏旁，黑色风衣被夜风向右鼓起。右手从风衣口袋中掏出一张折叠照片。城市灯光在@图1（顾南衣）面部形成冷色散射光，无主方向。镜头固定。3-7s：画面整体色温从冷蓝切换为暖黄，天台空间逐帧替换为@图5（旧宅客厅），午后阳光从左窗射入形成斜方光柱，尘埃在光柱中悬浮。画面带轻微胶片颗粒感，饱和度偏低。@图2（顾南衣-幼年）坐在木地板上，齐刘海下的眼睛向上看。@图3（父亲）蹲在@图2（顾南衣-幼年）面前，圆框眼镜在阳光下反射出一点亮光，右手递出一个棕色纸信封。镜头固定于中景。7-11s：@图3（父亲）将纸信封放在@图2（顾南衣-幼年）手中，@图2（顾南衣-幼年）双手接过，信封上有手写字迹清晰可读。@图3（父亲）嘴唇动作清晰说出一句话，面部表情认真但带柔和笑意。阳光光柱方向不变。镜头缓慢推进至信封与@图2（顾南衣-幼年）面部近景。11-15s：画面颗粒感加重，暖黄色调逐渐被冷蓝侵蚀，@图5（旧宅客厅）的家具轮廓变为模糊暗影。@图1（顾南衣）重新出现在@图4（天台），手中照片与记忆中的信封形成视觉对应，风衣被风拉向画面右侧。@图1（顾南衣）眼眶微红，下唇轻咬，目光从照片移向远处城市灯光。镜头固定。

3D写实动画，4K高清，细节丰富。人物面部稳定不变形，五官清晰，无穿模。现实与记忆的时间线用色温和颗粒感严格区分，过渡不硬切。记忆段信息（信封字迹）清晰可读。
```

</details>
