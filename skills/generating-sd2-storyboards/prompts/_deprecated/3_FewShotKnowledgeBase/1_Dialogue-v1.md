# Bucket: dialogue
v1.0

## 适用场景

- 双人或多人对白
- 对峙、盘问、施压、问答
- 需要稳定轴线和听者反应的段落

## 常见结构标签

- `two_person_confrontation`
- `group_dialogue`
- `listener_reaction`
- `table_barrier`
- `axis_sensitive`

## 注入重点

- 先建立空间关系，再进入说话者/听者节奏
- 保持轴线稳定，优先固定镜头或缓推
- 听者反应镜头不能丢
- 微表情要跟对白情绪同步，而不是机械念台词

## 密度约束（SD2 引擎安全）

- **单时间片优先一讲一听**，不要同时堆砌手部、眼神、身体、道具四种变化
- 多人场景除焦点两人外，其余角色默认保持**静止或极低动态**
- 肢体接触类对话（握手、触碰、拉扯）单独占用时间片，不与长口型或强情绪突变合并

## 代表性 few-shot

### `dialogue_two_person_lowkey_v1`

- `pattern_summary`: 双人低调对峙，主镜建立空间关系，再进入正反打与听者反应
- `camera_bias`: `["固定镜头", "缓慢推进"]`
- `must_cover`: `["axis_stability", "listener_reaction", "micro_expression"]`

<summary>example_prompt（完整三段式正例）</summary>

```
@图1（沈渡）为主角，青年男性，瘦削脸庞，深陷眼窝，身穿藏青色长衫。@图2（赵令仪）为对手角色，中年女性，发髻高挽，身着暗红色对襟褙子。场景为@图3（审讯室），封闭石室，仅一盏油灯悬于二人之间，暖黄灯光从顶部直射，照亮二人面部下半侧，眉弓以上沉入阴影。

0-4s：@图1（沈渡）坐在@图3（审讯室）的石凳上，双手平放于粗木桌面，指尖微微发白，目光低垂注视桌面裂缝。画面左侧的@图2（赵令仪）站立，双臂交叉于胸前，嘴角压着一丝冷笑。镜头固定，中景框住二人与桌面，绝对稳定。4-8s：@图2（赵令仪）身体前倾，右手掌按在桌面上发出沉闷响声，油灯火苗因气流轻微晃动，@图2（赵令仪）的面部暖光随之摇曳。@图1（沈渡）眼皮微抬，瞳孔收缩但身体纹丝不动，下颌肌肉紧绷。镜头固定不动。8-12s：@图1（沈渡）缓缓抬起头，与@图2（赵令仪）目光对视，微张双唇开口说话，声音平静但喉结微微滚动。@图2（赵令仪）后仰半步，双眉微拧，眼角细纹加深，冷笑逐渐消退。镜头缓慢推进至@图1（沈渡）面部近景。12-15s：@图2（赵令仪）右手缓缓从桌面收回，指尖在粗糙桌面划过，目光避开@图1（沈渡），偏向画面右下。@图1（沈渡）维持直视姿态，嘴角极轻微上扬但眼神冷硬。镜头固定，捕捉听者反应。

3D写实动画，4K高清，细节丰富。人物面部稳定不变形，五官清晰，无穿模。双人轴线关系全程保持一致，无跳轴。
```

### `dialogue_group_pressure_v1`

- `pattern_summary`: 多人场合中的单人承压，对话重心围绕核心角色逐步收紧
- `camera_bias`: `["固定镜头", "局部缓推"]`
- `must_cover`: `["reaction_contrast", "eye_focus_shift", "spatial_clarity"]`

<details>
<summary>example_prompt（完整三段式正例）</summary>

```
@图1（陆峥）为主角，青年军官，短发利落，着灰绿色军常服，领口扣紧。@图2（韩司令）坐于长桌正中，银发短寸，将星肩章。@图3（参谋甲）和@图4（参谋乙）分坐两侧。场景为@图5（作战会议室），长桌横贯画面，头顶日光灯管发出冷白光线，均匀照亮桌面与人物，无明显阴影方向。

0-4s：@图5（作战会议室）全景，长桌两侧坐满军官，画面正中的@图2（韩司令）双手交叉搁在桌面，面无表情注视画面右侧末端站立的@图1（陆峥）。@图3（参谋甲）和@图4（参谋乙）同时侧头看向@图1（陆峥）。镜头固定，全景构图。4-8s：@图1（陆峥）立正站姿，双手贴裤缝，喉结滚动一次后开口陈述，语速平稳但额头有细密汗珠。@图2（韩司令）右手食指在桌面无声敲击。镜头缓慢推进至@图1（陆峥）中景。8-12s：@图3（参谋甲）摇头并翻开面前的文件夹，纸页翻动声清晰。@图1（陆峥）目光短暂偏向@图3（参谋甲），随即回正继续陈述，嘴唇微微颤动。镜头固定于@图1（陆峥）近景，景深虚化背景众人。12-15s：@图2（韩司令）抬起右手示意静默，所有人停止动作。@图1（陆峥）闭嘴，下颌收紧，双眼微眯。@图2（韩司令）嘴角未动但目光从文件抬起直视@图1（陆峥）。镜头固定，双人关系中景。

3D写实动画，4K高清，细节丰富。人物面部稳定不变形，五官清晰，无穿模。多人物面部特征保持一致，严禁跳脸。
```

</details>
