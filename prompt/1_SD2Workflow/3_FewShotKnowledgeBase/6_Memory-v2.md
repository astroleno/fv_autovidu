# Bucket: memory
v2.0

## 适用场景

- 回忆、闪回
- 非现实时间线切入
- **独立的记忆 Block**——不含现实线

> **v2 关键修复**：memory 桶的示例必须是**独立记忆 Block**，不得在单个 prompt 内展示"现实→回忆→现实"的时间线切换。现实触发块属于 `emotion` / `transition`，纯记忆块属于 `memory`，现实回落块回到 `emotion` / `reveal`。

## 常见结构标签

- `flashback`
- `memory_fragment`
- `past_overlay`
- `recall_trigger`

## 常见视觉标签

- `non_physical_space`
- `flashback_texture`
- `desaturated`

## 注入重点

- **示例只描写记忆中的一段连续画面**，不含现实线切入/退出
- 允许非物理空间、暗角、柔焦、碎片化遮挡等表达
- 节奏通常短于现实段落
- 色温与现实线有明确区分（由 EditMap-SD2 在 Block 级定义）
- 现实→记忆的过渡和记忆→现实的过渡由**编排层**在 Block 之间处理，不在单个 memory Block 内实现

## 代表性 few-shot

### `memory_pure_fragment_v1`（v2 新增·替代 v1 的 memory_fragment_cold_v1）

- `pattern_summary`: 纯记忆空间内的一段连续画面，不含现实线
- `camera_bias`: `["缓慢横移", "固定镜头"]`
- `must_cover`: `["memory_texture", "fact_delivery", "timeline_separation"]`

> **替代说明**：v1 的 `memory_fragment_cold_v1` 在单个 prompt 内包含了"现实书房→记忆白光场→回到书房"的完整切换，违反了 SD2 引擎的单镜头连续视频约束。v2 将其拆分：现实触发部分归入 `emotion` 桶，纯记忆部分作为本示例。

<details>
<summary>example_prompt（完整三段式正例）</summary>

```
@图1（许砚-少年）为记忆中的主角形象，少年面庞，神情专注。场景为非物理背景，纯白过曝光场，边缘渐暗，带柔焦和轻微胶片颗粒感。光线从四周均匀散射，无明确主光源方向。光线稳定。

0-5s：纯白过曝光场中，@图1（许砚-少年）从画面左侧缓步走入，身体轮廓边缘在过曝背景中微微发光，面部特征清晰但周围空间无物理参照。@图1（许砚-少年）低头看着右手，右手握着一枚旧钥匙。轻微柔焦笼罩整个画面。镜头缓慢横移跟随@图1（许砚-少年）。5-10s：@图1（许砚-少年）停下脚步，抬起右手将钥匙举到面前，钥匙齿槽在散射光中清晰可辨。@图1（许砚-少年）嘴角微微上扬，目光柔和注视钥匙。纯白背景边缘保持柔和暗角。镜头固定于@图1（许砚-少年）中景。10-15s：@图1（许砚-少年）将钥匙收回掌心，右手在胸前轻轻合拢。抬头看向画面正前方，表情从微笑渐变为平静。纯白过曝光场、柔焦与边缘暗角保持稳定。镜头固定。

3D写实动画，4K高清，细节丰富。人物面部稳定不变形，五官清晰，无穿模。记忆段柔焦与低饱和全程一致。画面边缘暗角过渡平滑。
```

</details>

**structural_notes**（v2 新增）:
- **这是纯记忆 Block**——没有现实线切入/退出，只描写记忆中的一段连续画面
- 三段式：入画（缓步走入）→ 信息呈现（钥匙特写）→ 情感收束（收回掌心+平静）
- 非物理空间的表达：纯白过曝光场 + 边缘暗角 + 柔焦 + 无物理参照
- 钥匙是"记忆中的信息载体"——道具细节（齿槽）必须清晰可辨
- 暗角与柔焦在全段保持稳定——记忆感来自持续质感，而不是镜头内 look 变化

**anti_patterns**（v2 关键修复）:
- **绝对禁止**：在单个 memory prompt 内包含"现实→记忆→现实"的切换——这违反 SD2 引擎单镜头约束
- **绝对禁止**：在 memory Block 中描写当前现实时间线的任何元素（现实中的角色、现实中的场景）
- 不要迁移"纯白过曝光场"作为所有记忆的默认表达——记忆可以是具体场景（旧宅、学校操场等），非物理空间只是选项之一
- 不要迁移"钥匙"——信息载体要匹配实际 Block 中的道具

### `memory_scene_recall_v1`（v2 新增·替代 v1 的 memory_reveal_insert_v1）

- `pattern_summary`: 在具体场景中发生的回忆片段，有物理空间参照，承载信息揭示
- `camera_bias`: `["固定镜头", "缓慢推进"]`
- `must_cover`: `["fact_delivery", "memory_texture", "emotional_echo"]`

> **替代说明**：v1 的 `memory_reveal_insert_v1` 包含了天台（现实）→旧宅（记忆）→天台（现实）的切换。v2 只保留旧宅（记忆）部分。

<details>
<summary>example_prompt（完整三段式正例）</summary>

```
@图1（顾南衣-幼年）为记忆中的主角形象，七八岁女童，神情专注。@图2（父亲）为记忆中出现的角色，中年男性，目光温和。场景为@图3（旧宅客厅），午后阳光从左侧窗户射入，暖黄色调，画面带轻微胶片颗粒感，饱和度偏低。光线稳定。

0-4s：@图3（旧宅客厅）中景，午后阳光从左窗射入形成斜方光柱，尘埃在光柱中悬浮。@图1（顾南衣-幼年）坐在木地板上，双眼向上看。@图2（父亲）蹲在@图1（顾南衣-幼年）面前，面部轮廓在阳光下反射出一点亮光，右手拿着一个棕色纸信封。镜头固定于中景。4-8s：@图2（父亲）将纸信封递向@图1（顾南衣-幼年），@图1（顾南衣-幼年）双手接过，信封上有手写字迹清晰可读。@图2（父亲）嘴唇动作清晰说出一句话，面部表情认真但带柔和笑意。阳光光柱方向不变。镜头缓慢推进至信封与@图1（顾南衣-幼年）面部近景。8-12s：@图1（顾南衣-幼年）低头看信封上的字迹，双手拇指轻轻摩挲信封表面。@图2（父亲）右手伸出轻轻拍了一下@图1（顾南衣-幼年）的头顶。@图1（顾南衣-幼年）抬头看@图2（父亲），嘴角上扬露出牙齿。阳光在@图1（顾南衣-幼年）面部边缘形成稳定的暖光轮廓。镜头固定。12-15s：@图2（父亲）和@图1（顾南衣-幼年）维持近乎静止的姿态，@图1（顾南衣-幼年）嘴角停留在极轻的上扬后缓慢收回。画面保持稳定的胶片颗粒感、低饱和暖光和边缘柔化，不发生进一步色温或质感变化。镜头固定。

3D写实动画，4K高清，细节丰富。人物面部稳定不变形，五官清晰，无穿模。记忆段暖光、低饱和与胶片颗粒感全程一致。信息载体（信封字迹）清晰可读。
```

</details>

**structural_notes**（v2 新增）:
- **这是纯记忆 Block**——在一个具体的过去场景中发生，有物理空间但带记忆质感（胶片颗粒+低饱和）
- 四段式：场景建立 → 信息交付（信封传递）→ 情感互动（拍头+微笑）→ 静止收尾（动作停稳）
- 信息载体（信封字迹）必须"清晰可读"——记忆中的关键信息不能被柔焦模糊
- 最后一段是记忆 Block 内部的静止收尾——给编排层在下一个 Block 切回现实预留落点

**anti_patterns**（v2 关键修复）:
- **绝对禁止**：在这个 Block 中切回现实时间线（天台/成年顾南衣等）——现实线由编排层在下一个 Block 处理
- 不要迁移"旧宅""信封""圆框眼镜"——场景和道具要匹配实际
- 不要在所有记忆场景都用"暖黄色调"——记忆的色温由上游 `artStyle` 和 `visuals` 定义
- 不要在最后一段做镜头内质感/色温渐变——颗粒、饱和度、色温变化只能在 Block 切换时完成
