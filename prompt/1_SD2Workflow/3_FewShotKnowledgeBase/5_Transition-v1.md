# Bucket: transition
v1.0

## 适用场景

- 建立环境
- 角色进场或离场
- 段落桥接、时空过渡、节奏呼吸
- 非高潮但必须交代空间关系的连接段

## 常见结构标签

- `entry_exit`
- `establishing`
- `bridge_beat`
- `approach`
- `spatial_reset`

## 注入重点

- 优先交代空间、方向和段落功能
- 情绪表达从属于段落连接，不抢主戏
- 运镜以固定、缓移、缓推为主，避免无意义炫技

## 密度约束（反脑补）

- **建立段只交代空间主结构、方向线、入口出口**，不得自行补充动物、天气、装饰物、次要道具
- 环境细节仅限 `assets_required.scenes[]` 白名单中已有的元素和 `location.place` 描述范围内的物理结构
- 若原始剧本未提及的环境装饰（灯笼、灰猫、排水沟、残光等）出现在 few-shot 示例中，**严禁迁移到当前 prompt**

## 代表性 few-shot

### `transition_establish_lowkey_v1`

- `pattern_summary`: 低调氛围建立，先交代空间，再引出角色或下一动作
- `camera_bias`: `["固定镜头", "缓慢横移"]`
- `must_cover`: `["spatial_clarity", "atmosphere", "entry_hint"]`

<details>
<summary>example_prompt（完整三段式正例）</summary>

```
@图1（萧远）为即将进场的角色，暂不出镜。场景为@图2（古巷），黎明前，窄长石板巷道，两侧是斑驳白墙与黑瓦檐口，远端巷口透出极淡的灰蓝天光。地面石板缝隙残留昨夜雨水，湿漉漉反光。一盏将灭的纸灯笼挂在左墙铁钩上，发出昏黄残光。

0-5s：@图2（古巷）全景，镜头从巷道左墙的纸灯笼开始，灯笼内烛火微弱跳动即将熄灭，黄光在白墙上投出摇曳光圈。灯笼下方石板地面湿润，反射出暗淡暖光。镜头缓慢向右横移。5-10s：镜头继续横移，扫过巷道中段的排水沟渠，渠中有细流缓缓流淌。右墙根部一只灰猫蜷缩在门槛台阶上，耳朵偶尔转动。远端巷口的灰蓝天光逐渐扩大占据画面上方。镜头保持缓慢横移。10-14s：镜头横移至巷口方向停稳，远端巷口形成明亮的灰蓝色矩形光框，石板巷道形成纵深引导线。一个模糊的人影从巷口光框中出现，逆光轮廓无法辨认面部，正缓步走入巷道。镜头固定。

3D写实动画，4K高清，细节丰富。环境空间稳定无闪烁，石板与墙面材质纹理清晰。极微弱的自然光影呼吸感。
```

</details>

### `transition_exit_bridge_v1`

- `pattern_summary`: 角色完成一个动作后离开当前段落，镜头承担接续功能
- `camera_bias`: `["固定镜头", "轻微跟随"]`
- `must_cover`: `["exit_direction", "bridge_state", "handoff"]`

<details>
<summary>example_prompt（完整三段式正例）</summary>

```
@图1（叶青鸾）为主角，年轻女性，长发半束，额头有汗渍，身穿深灰色骑装，左臂缠绷带。场景为@图2（驿站门前），傍晚，夕阳从画面右侧低角度射入，暖橙色长影铺满黄土地面。驿站木门半掩，门楣上方挂一面褪色布幌。

0-5s：@图2（驿站门前）中景，@图1（叶青鸾）站在门口，右手扶着门框，目光向画面左侧远眺。夕阳暖光从右侧照亮@图1（叶青鸾）面部右半侧，左半脸被门框阴影遮挡。骑装上的扣件反射暖色高光。镜头固定。5-9s：@图1（叶青鸾）松开门框，转身背对镜头向画面左侧走去，长发因转身甩动后落在肩上。左臂绷带在夕阳下呈淡橙色。影子在黄土地面拉长至画面左下角。镜头轻微跟随@图1（叶青鸾）移动方向。9-13s：@图1（叶青鸾）背影逐渐走向画面左侧远端，身形在夕阳逆光中变为暖色剪影边缘。驿站门前的布幌被风轻轻鼓起。镜头固定，@图1（叶青鸾）背影缩小至画面左三分之一处。

3D写实动画，4K高清，细节丰富。人物面部稳定不变形，五官清晰，无穿模。夕阳色温全程一致，无突变。
```

</details>
