# Bucket: transition
v2.0

## 适用场景

- 建立环境
- 角色进场或离场
- 段落桥接、时空过渡、节奏呼吸
- 非高潮但必须交代空间关系的连接段

## 常见结构标签

- `entry_exit`
- `establishing`
- `bridge_beat`
- `approach`
- `spatial_reset`

## 注入重点

- 优先交代空间、方向和段落功能
- 情绪表达从属于段落连接，不抢主戏
- 运镜以固定、缓移、缓推为主，避免无意义炫技

## 密度约束（反脑补·v2 强化）

- **建立段只交代空间主结构、入口出口、方向线**，不得自行补充动物、天气、装饰物、次要道具
- 环境细节仅限 `assets_required.scenes[]` 白名单中已有的元素和 `location.place` 描述范围内的物理结构
- 若原始剧本未提及的环境装饰出现在 few-shot 示例中，**严禁迁移到当前 prompt**
- **v2 强化**：每个示例的 `anti_patterns` 中会显式列出不可迁移的装饰性元素

## 代表性 few-shot

### `transition_establish_lowkey_v2`

- `pattern_summary`: 低调氛围建立，先交代空间，再引出角色或下一动作
- `camera_bias`: `["固定镜头", "缓慢横移"]`
- `must_cover`: `["spatial_clarity", "atmosphere", "entry_hint"]`

<details>
<summary>example_prompt（完整三段式正例）</summary>

```
@图1（萧远）为即将进场的角色，暂不出镜。场景为@图2（古巷），黎明前，窄长石板巷道，两侧是斑驳白墙与黑瓦檐口，远端巷口透出极淡的灰蓝天光。地面石板缝隙残留昨夜雨水，湿漉漉反光。光线稳定。

0-5s：@图2（古巷）全景，镜头从巷道左墙开始，墙面斑驳。地面石板湿润，反射出暗淡光线。镜头缓慢向右横移。5-10s：镜头继续横移，扫过巷道中段。远端巷口的灰蓝天光逐渐扩大占据画面上方。镜头保持缓慢横移。10-14s：镜头横移至巷口方向停稳，远端巷口形成明亮的灰蓝色矩形光框，石板巷道形成纵深引导线。一个模糊的人影从巷口光框中出现，逆光轮廓无法辨认面部，正缓步走入巷道。镜头固定。

3D写实动画，4K高清，细节丰富。环境空间稳定无闪烁，石板与墙面材质纹理清晰。极微弱的自然光影呼吸感。
```

</details>

**structural_notes**（v2 新增）:
- 建立段三步：空间入口（墙面/地面）→ 空间纵深（巷道/天光）→ 角色入画（逆光人影）
- 横移运镜只有一个功能：引导视线从"入口"移到"出口"
- 最后人影是"暗示"而非"呈现"——不揭示身份，只制造期待
- 全程零角色行为描写——纯环境建立

**anti_patterns**（v2 强化·关键修复）:
- **禁止迁移**：`纸灯笼`、`灰猫`、`排水沟渠`、`残光`——v1 示例中包含的这些装饰性元素**不属于空间主结构**，不得在其他场景中使用
- **禁止迁移**：`昨夜雨水`、`湿漉漉`——天气状态要匹配实际 Block，不是默认下雨
- 不要在所有建立段都用横移——横移适用于线性空间（巷道/走廊），非线性空间（广场/客厅）应用固定全景
- 建立段的环境细节**只描写空间主结构**（墙、地、天花板、出入口），不加装饰

### `transition_exit_bridge_v2`

- `pattern_summary`: 角色完成一个动作后离开当前段落，镜头承担接续功能
- `camera_bias`: `["固定镜头", "轻微跟随"]`
- `must_cover`: `["exit_direction", "bridge_state", "handoff"]`

<details>
<summary>example_prompt（完整三段式正例）</summary>

```
@图1（叶青鸾）为主角，年轻女性，额头有汗渍，左臂带伤。场景为@图2（驿站门前），傍晚，夕阳从画面右侧低角度射入，暖橙色长影铺满黄土地面。驿站木门半掩。光线稳定。

0-5s：@图2（驿站门前）中景，@图1（叶青鸾）站在门口，右手扶着门框，目光向画面左侧远眺。夕阳暖光从右侧照亮@图1（叶青鸾）面部右半侧，左半脸被门框阴影遮挡。镜头固定。5-9s：@图1（叶青鸾）松开门框，转身背对镜头向画面左侧走去，人物轮廓因转身带出短暂摆动。左臂包扎处在夕阳下呈淡橙色。影子在黄土地面拉长至画面左下角。镜头轻微跟随@图1（叶青鸾）移动方向。9-13s：@图1（叶青鸾）背影逐渐走向画面左侧远端，身形在夕阳逆光中变为暖色剪影边缘。镜头固定，@图1（叶青鸾）背影缩小至画面左三分之一处。

3D写实动画，4K高清，细节丰富。人物面部稳定不变形，五官清晰，无穿模。夕阳色温全程一致，无突变。
```

</details>

**structural_notes**（v2 新增）:
- 离场三步：驻足远眺（最后的停留）→ 转身离开（行动决定）→ 剪影远去（情感余韵）
- 离场方向要清晰——从画面右侧走向左侧，影子拉长方向与运动方向一致
- 最后一段是"情感交接"——背影剪影是段落结束的视觉信号
- 跟随运镜只在中间段使用，最后固定，让角色"走出"镜头

**anti_patterns**（v2 新增）:
- 不要迁移"驿站""左臂包扎处""黄土地面"——场景和角色状态要匹配实际
- 不要迁移"夕阳暖橙"——色温要匹配实际 artStyle 和 lighting_state
- 不要在所有离场都写"剪影远去"——这是情感型离场，功能型离场不需要
- 不要在离场段加新角色或新事件——离场是收束，不是展开
