# Bucket: emotion
v2.0

## 适用场景

- 单人独角戏
- 沉默反应、凝视、自我消化
- 情绪递进但无大动作场面
- 情绪转折点（平静→被击中的瞬间）（v2 新增）

## 常见结构标签

- `single_subject`
- `gaze_hold`
- `material_anchor`
- `interior_pressure`
- `emotion_turning`（v2 新增）

## 常见补强目标

- `micro_expression`
- `breathing_detail`
- `eye_focus_shift`
- `restraint`
- `slow_push`
- `silence_density`（v2 新增）
- `protagonist_reaction`（v2 新增）

## 注入重点

- 节奏偏克制，优先 2-4 个时间片
- 用手部、呼吸、视线细节承载情绪变化——**纯物理描述，禁止比喻**
- 运镜以固定镜头、缓推、轻微漂移为主
- 重点不是"发生了什么"，而是"角色如何感受"
- 沉默段描写密度 ≥ 台词段

## 代表性 few-shot

### `emotion_awakening_lowkey_v2`

- `pattern_summary`: 单主体觉醒，先建立压迫氛围，再推进到认知变化
- `camera_bias`: `["缓慢推进", "固定镜头"]`
- `must_cover`: `["micro_expression", "material_interaction", "slow_push"]`

<details>
<summary>example_prompt（完整三段式正例）</summary>

```
@图1（秦狩）为主角，年轻皇子，神情警觉。场景为@图2（寝宫），夜间灯火，古色古香的皇家寝宫，幽暗压抑，烛火摇曳。一束冷青色窄光从画面左侧切入，照亮面部左半侧，右半脸沉入深青浓重阴影。光线稳定。

0-5s：@图2（寝宫）全景，帷幔垂帘虚化前景，锦缎软榻位于画面中央偏右，@图1（秦狩）在软榻上猛然惊坐起，被面因身体重压留有深深褶皱。帷幔轻微晃动。镜头从寝宫全景缓慢向@图1（秦狩）推进。5-9s：@图1（秦狩）惊恐地环顾四周陌生的环境，眼神快速扫视左右，嘴唇微张，颧骨上冷白高光硬切分明，面部肌肉紧绷。镜头固定，绝对稳定。9-15s：@图1（秦狩）缓缓伸出双手，低头凝视自己年轻的双手，冷白高光落在手背骨节上，指节清晰，瞳孔微微颤动，表情从震惊渐变为不可置信。镜头缓慢推进至@图1（秦狩）双手与面部的近景。

3D写实动画，4K高清，细节丰富，粗颗粒胶片质感。人物面部稳定不变形，五官清晰，无穿模，面部表情自然过渡。
```

</details>

**structural_notes**（v2 新增）:
- 三段递进：环境压迫（全景）→ 意识模糊（近景扫视）→ 认知变化（手部特写）
- 第三段（9-15s）是最重要的情绪落点，时长最长，描写密度最高
- 手部凝视是"觉醒"模式的标志性动作——适用于所有"发现自己状态变化"的场景
- 只有三个时间片，节奏极克制

**anti_patterns**（v2 新增）:
- 不要迁移"冷青色窄光"——光源要匹配实际 artStyle
- 不要迁移"帷幔""锦缎软榻"——这是皇家场景的环境细节
- 不要在非觉醒场景中套用"端详双手"的动作
- 不要写"皮肤突然发白/发青"——避免铁律 2 的皮肤变色禁忌

### `emotion_silent_break_v2`

- `pattern_summary`: 角色强忍情绪，表面静止，内部断裂逐步外显
- `camera_bias`: `["固定镜头", "轻微漂移"]`
- `must_cover`: `["breathing_detail", "eye_focus_shift", "restraint", "silence_density"]`

<details>
<summary>example_prompt（完整三段式正例）</summary>

```
@图1（苏晚）为主角，年轻女性，面色疲惫。场景为@图2（病房），深夜无人的单人病房，窗外城市灯光透过百叶窗在墙面投下均匀横纹，冷蓝色调。床头监护仪屏幕微光映照@图1（苏晚）面部右侧。光线稳定。

0-5s：@图1（苏晚）坐在@图2（病房）床沿，双脚悬空，双手握着一张折叠的信纸，手指关节发白。目光固定在信纸上不动，胸廓极缓慢起伏，呼吸浅而克制。镜头固定，中景侧面取景。5-10s：@图1（苏晚）右手拇指无意识地摩挲信纸折痕，纸面因反复折叠已起毛边。视线从信纸缓缓移开，偏向窗外百叶窗的光纹，瞳孔焦距变远发散，嘴唇紧闭两侧肌肉微微凹陷。监护仪微光在面部右侧持续脉动。镜头轻微向右漂移。10-15s：@图1（苏晚）下唇突然颤抖一次随即咬住，鼻翼翕动，眼眶泛红但无泪水落下。左手缓缓松开信纸，信纸边角轻微卷曲。右手抬起覆盖住口鼻，手指微微颤抖。镜头固定于面部近景。

3D写实动画，4K高清，细节丰富。人物面部稳定不变形，五官清晰，无穿模。面部微表情过渡自然，无跳帧式突变。
```

</details>

**structural_notes**（v2 新增）:
- 克制→外溢三段式：静态握持（克制阶段）→ 视线游移（松动阶段）→ 下唇颤抖+攥拳（断裂阶段）
- 物理锚点是"信纸"——手指摩挲折痕、纸面毛边、信纸卷曲，都在用物件承载情绪
- "眼眶泛红"是铁律 2 的唯一例外——仅用于哭泣前兆
- 三个时间片时长等分（各 5s），给每个阶段充足的"呼吸"空间

**anti_patterns**（v2 新增）:
- 不要迁移"病房""监护仪""百叶窗"——这是特定场景细节
- 不要迁移"信纸"——物理锚点要匹配实际 Block 中的道具
- 不要在非哭泣场景中使用"眼眶泛红"
- 不要缩短时间片到 2-3s——克制型情绪需要慢节奏呼吸

### `emotion_turning_point_v1`（v2 新增）

- `pattern_summary`: 角色从平静被一个信息/事件击中，情绪突变的瞬间
- `camera_bias`: `["固定镜头", "快速推进"]`
- `must_cover`: `["face_change", "recognition_shift", "timing_punch"]`

<details>
<summary>example_prompt（完整三段式正例）</summary>

```
@图1（江铃）为主角，年轻女性，神情平静。场景为@图2（办公室），白天，日光灯均匀照亮整个空间，冷白色调。光线稳定。

0-5s：@图1（江铃）坐在@图2（办公室）工位前，双手放在键盘上，面部表情平静放松，嘴角带着日常的淡定弧度。目光注视屏幕，手指偶尔轻敲键盘。镜头固定于@图1（江铃）中景。5-8s：@图1（江铃）右手从键盘移开拿起手机，目光移到手机屏幕上，瞳孔在 0.5 秒内骤缩，嘴角的弧度消失，整张脸在一瞬间从松弛切换为僵硬。手指握紧手机边框，指节微微发白。镜头快速推进至@图1（江铃）面部近景。8-12s：@图1（江铃）嘴唇微张随即紧闭，下颌肌肉跳动一次，喉结上下滚动。双眼仍盯着手机屏幕，但瞳孔焦距变化——从聚焦变为发散再回到聚焦。左手无意识地攥紧桌面上的一支笔。镜头固定于@图1（江铃）面部特写。12-15s：@图1（江铃）缓慢放下手机，屏幕朝下扣在桌面上。双手收回放在膝盖上，手指微颤。目光从手机位置移开，看向窗外，呼吸加深一次后恢复平稳。面部肌肉逐渐从僵硬放松为一种空洞的平静。镜头固定。

3D写实动画，4K高清，细节丰富。人物面部稳定不变形，五官清晰，无穿模。情绪转折瞬间过渡自然。
```

</details>

**structural_notes**（v2 新增）:
- 四段递进：日常平静（建立基线）→ 信息击中（0.5s 内表情断裂）→ 消化吸收（物理反应外溢）→ 重新压制（假平静）
- 情绪转折点的关键是**第二段的速度**——5-8s 内完成从松弛到僵硬的切换，快速推进运镜强化冲击
- 第三段（8-12s）是核心描写段——瞳孔焦距变化、下颌跳动、无意识攥笔
- 最后一段不是"恢复正常"而是"假装正常"——空洞的平静比真正的平静更有张力

**anti_patterns**（v2 新增）:
- 不要迁移"手机""键盘""办公室"——触发物和场景要匹配实际 Block
- 不要写"内心翻涌""强忍泪水"——这是虚词，要转化为物理动作
- 不要在第二段（击中瞬间）堆太多动作——一个清晰的表情断裂即可
- 不要在最后一段做大幅度情绪释放——情绪转折点的价值在于"被压住"的张力
