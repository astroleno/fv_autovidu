# Bucket: reveal
v2.0

## 适用场景

- 真相揭示
- 身份确认
- 觉醒、认知反转
- 权力关系突然变化
- 道具驱动的真相揭示（v2 新增）

## 常见结构标签

- `awakening`
- `truth_drop`
- `power_reversal`
- `recognition`
- `beat_escalation`
- `prop_driven_reveal`（v2 新增）

## 注入重点

- 节奏通常为"铺垫 → 认知变化 → 结果"
- 允许在中后段加快推进，强化认知冲击
- 重点是表情与理解的变化，而不是额外动作堆砌

## 代表性 few-shot

### `reveal_identity_turn_v2`

- `pattern_summary`: 从信息铺垫进入表情断裂，再落到新关系的成立
- `camera_bias`: `["缓慢推进", "固定镜头"]`
- `must_cover`: `["recognition_shift", "face_change", "status_reset"]`

<details>
<summary>example_prompt（完整三段式正例）</summary>

```
@图1（姜辞）为主角，青年男性，面容清秀但眼神锐利。@图2（周老太爷）为次要角色，年迈男性，拄拐杖。场景为@图3（祠堂），夜间，正堂牌位高悬，两列红烛提供暖橙侧光，梁柱在地面投下长条阴影。光线稳定。

0-4s：@图3（祠堂）中景，@图2（周老太爷）背对镜头站于牌位前，拐杖顶端撑地。画面右侧的@图1（姜辞）站在门槛处，目光平静扫过牌位上的字迹，双手自然垂于身侧。红烛暖光从两侧照亮二人面部，鼻梁两侧形成对称明暗。镜头固定。4-8s：@图2（周老太爷）缓缓转身，拐杖在石地发出一声沉响，面部从阴影转入烛光，皱纹深刻。@图2（周老太爷）嘴唇微张说出一句话。@图1（姜辞）瞳孔骤缩，眼角肌肉跳动一次，嘴角原有的淡定弧度消失，整张脸在半秒内从平静切换为僵硬。镜头缓慢推进至@图1（姜辞）面部。8-12s：@图1（姜辞）下意识后退半步，右脚踩在门槛边缘，身体重心轻微晃动。双眼直视@图2（周老太爷），嘴唇抿紧后又松开，喉结上下滚动一次，额头青筋微隐。红烛火苗因穿堂风轻晃，面部光影随之交替。镜头固定于近景。12-16s：@图1（姜辞）缓缓挺直腰背，下颌微收，目光从震惊重新凝聚为锐利。右手缓慢抬起握成拳，指节泛白。@图2（周老太爷）拄拐杖的手微微颤抖，目光向下避开@图1（姜辞）的注视。镜头固定，双人关系中景。

3D写实动画，4K高清，细节丰富。人物面部稳定不变形，五官清晰，无穿模。面部表情断裂过渡自然，不出现跳帧突变。
```

</details>

**structural_notes**（v2 新增）:
- 四段式揭示：建立（静态对峙）→ 信息投递（转身+一句话）→ 消化吸收（后退+表情变化）→ 新关系成立（挺直腰+攥拳 vs 颤抖+回避）
- 表情断裂发生在第二段末尾——"半秒内从平静切换为僵硬"是关键技法
- 最后一段展示的是**关系翻转**而非单纯的情绪反应——主角凝聚力量 vs 老太爷开始颤抖
- 受力方（姜辞）的镜头时长和描写密度始终高于施力方（周老太爷）

**anti_patterns**（v2 新增）:
- 不要迁移"祠堂""牌位""红烛"——场景细节要匹配实际 Block
- 不要迁移"门槛""石地"——物理环境要匹配实际
- 不要在所有揭示场景都写"后退半步"——这是特定的惊讶反应
- "额头青筋微隐"属于铁律 2 的边缘情况——青筋是物理结构不是皮肤变色，可保留

### `reveal_power_flip_v2`

- `pattern_summary`: 原本弱势的一方在一句话或一个动作后取得上风
- `camera_bias`: `["固定镜头", "快速推进"]`
- `must_cover`: `["hierarchy_flip", "reaction_contrast", "timing_punch"]`

<details>
<summary>example_prompt（完整三段式正例）</summary>

```
@图1（林朝歌）为主角，年轻女性，面带淤伤，双手铐链。@图2（谢监察）为对手角色，中年男性，方脸国字。场景为@图3（刑讯阁），石砌密室，头顶单灯投射硬顶光，@图1（林朝歌）面部下方形成浓重阴影，@图2（谢监察）大半身处于侧方弱光中。光线稳定。

0-3s：@图3（刑讯阁）中景，画面左侧的@图1（林朝歌）坐在木椅上，铐链垂于膝前，身体微微前倾，目光低垂。画面右侧的@图2（谢监察）双手背后站立，下巴微扬，俯视@图1（林朝歌）。镜头固定。3-7s：@图2（谢监察）右手从背后伸出，将一张纸拍在@图1（林朝歌）面前的石台上，纸面在顶光下泛白。@图1（林朝歌）视线缓缓移到纸面，瞳孔微动读取内容，面部肌肉没有任何变化。镜头固定。7-10s：@图1（林朝歌）嘴角极缓慢上扬，露出一个不带温度的笑容，抬头直视@图2（谢监察），铐链因手腕微转发出金属碰撞声。@图2（谢监察）笑容凝固，眉心肌肉不自觉收紧。镜头快速推进至@图1（林朝歌）面部特写。10-14s：@图1（林朝歌）平静开口说出一句话，嘴唇动作清晰。@图2（谢监察）后退一步，右手不自觉触碰腰间银牌，手指发颤。顶光下@图2（谢监察）额头出现汗珠反光。镜头固定于双人关系中景，@图1（林朝歌）居画面中心。

3D写实动画，4K高清，细节丰富。人物面部稳定不变形，五官清晰，无穿模。权力关系翻转的表情节奏精确，无机械切换感。
```

</details>

**structural_notes**（v2 新增）:
- 权力翻转四段式：弱势建立 → 施压（纸+拍桌）→ **翻转瞬间**（冷笑+直视）→ 新关系成立（平静说话 vs 后退发颤）
- 翻转的 timing 精度是关键——7-10s 的"嘴角极缓慢上扬"用了 3 秒，足够观众感受到权力转移
- 快速推进运镜只在翻转瞬间使用，强化冲击，其余全部固定
- 翻转后，弱势方变为画面中心位——构图即权力

**anti_patterns**（v2 新增）:
- 不要迁移"铐链""石砌密室""顶光"——场景和道具要匹配实际
- 不要在非翻转场景使用快速推进——这是高价值运镜，只在关键瞬间使用
- 不要写"汗珠从额头滑下"（过度描写），"额头出现汗珠反光"足够

### `reveal_prop_driven_v1`（v2 新增）

- `pattern_summary`: 以道具（信件/手机/证据）作为真相的物理载体，镜头在道具与表情之间切换
- `camera_bias`: `["固定镜头", "缓慢推进"]`
- `must_cover`: `["fact_delivery", "recognition_shift", "material_anchor"]`

<details>
<summary>example_prompt（完整三段式正例）</summary>

```
@图1（何念）为主角，年轻女性，神情克制。@图2（旧照片）为关键道具。场景为@图3（储物间），昏暗，头顶裸灯泡发出暖黄光，照亮@图1（何念）头顶和肩部，面部大部分处于由上而下的硬光阴影中。光线稳定。

0-4s：@图1（何念）蹲在@图3（储物间）角落的纸箱旁，右手翻开纸箱盖，左手从纸箱中拿出@图2（旧照片），照片正面朝下。@图1（何念）面部表情平静专注，嘴唇轻闭。头顶灯泡暖光在照片背面形成柔和反射。镜头固定于@图1（何念）中景。4-8s：@图1（何念）缓缓翻转@图2（旧照片），照片正面逐渐显露。镜头缓慢推进至@图2（旧照片）与@图1（何念）手部的近景，照片内容在灯泡光下清晰可辨。@图1（何念）翻转照片的手指动作突然停顿，拇指按在照片边缘不动。8-11s：@图1（何念）面部大特写，瞳孔从照片上移开又迅速回到照片，眉心微皱，嘴唇张开又合上，喉结缓慢滚动。呼吸从平稳变为一次深吸气后屏住。头顶灯泡光在面部形成从上向下的硬边阴影。镜头固定于@图1（何念）面部特写。11-15s：@图1（何念）缓缓将@图2（旧照片）翻扣在膝盖上，双手覆盖照片，手指微颤。目光从照片位置抬起看向前方虚空，瞳孔焦距发散。嘴唇紧抿，下颌两侧肌肉凸起。镜头缓慢后退至@图1（何念）中景。

3D写实动画，4K高清，细节丰富。人物面部稳定不变形，五官清晰，无穿模。道具细节清晰可辨。
```

</details>

**structural_notes**（v2 新增）:
- 道具揭示四段式：发现道具 → 道具信息显露 → 表情消化 → 压制/隐藏
- 镜头在"道具特写"和"表情特写"之间切换——道具承载事实，表情承载情绪
- 第二段（4-8s）的关键是"手指停顿"——物理上的停滞传达认知上的冲击
- 最后翻扣照片是"压制"的物理化——不是撕毁/扔掉（过度反应），而是覆盖/隐藏

**anti_patterns**（v2 新增）:
- 不要迁移"储物间""纸箱""裸灯泡"——场景和光源要匹配实际
- 不要迁移"旧照片"——道具要匹配实际 Block 中的 props
- 不要在道具揭示场景中写大幅度肢体反应（摔东西/站起来/跑出去）——道具揭示的力量在于克制
- 不要写"照片内容令人震惊"——这是叙述而非描写
