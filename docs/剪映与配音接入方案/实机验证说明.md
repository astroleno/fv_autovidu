# 剪映草稿实机验证说明

本文件配合「剪映草稿导出」功能使用：剪映桌面版对 `draft_info.json` 的字段要求可能随版本变化，**必须在目标环境实机验证**。

## 建议步骤

1. 在分镜板为若干镜头**选定视频**，使用 **导出 → 剪映草稿 ZIP**。
2. 解压 ZIP，确认含 `draft_info.json`、`draft_meta_info.json`、`materials/` 下视频文件。
3. 将整个草稿文件夹复制到本机剪映草稿目录（macOS 常见候选见 `GET /api/export/jianying-draft/path` 返回值），或通过剪映「导入」能力尝试加载（以实际剪映版本为准）。
4. 若无法打开：记录剪映版本号与报错现象，调整 `web/server/services/jianying_protocol.py` 中的模板版本与 `jianying_service.build_draft_info` 的字段结构。

## 后端协议版本

- 常量：`jianying_protocol.PROTOCOL_TEMPLATE_VERSION`
- 写入位置：`draft_info.json` 的 `draft_template_version` 字段
